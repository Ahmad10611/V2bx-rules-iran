
# V2bx-rules-iran

## معرفی پروژه

**V2bx-rules-iran** یک پروژه برای مدیریت و به‌روزرسانی خودکار قوانین V2bx است که به طور خاص برای ایران تنظیم شده است. این پروژه به شما امکان می‌دهد تا قوانین فایروال و مسیریابی مربوط به V2bx را به طور خودکار به‌روزرسانی و مدیریت کنید.

## ویژگی‌ها

- **به‌روزرسانی خودکار قوانین V2bx**: این پروژه به طور خودکار قوانین مربوط به ایران را دانلود و به‌روزرسانی می‌کند.
- **مدیریت سرویس‌ها با استفاده از systemd**: با راه‌اندازی این پروژه، اسکریپت‌های مربوط به به‌روزرسانی به صورت خودکار اجرا می‌شوند و نیازی به مدیریت دستی آن‌ها نیست.
- **کنترل و نظارت ساده**: امکان نظارت بر سرویس‌ها و بررسی وضعیت آن‌ها از طریق دستورات systemd وجود دارد.

## نحوه نصب

برای نصب پروژه کافی است دستور زیر را در سرور خود اجرا کنید:

```bash
curl -s https://raw.githubusercontent.com/Ahmad10611/V2bx-rules-iran/master/install.sh | bash
```

این دستور به صورت خودکار تمام مراحل نصب را انجام می‌دهد، از جمله:
1. دانلود و نصب ابزارهای مورد نیاز (مثل `dos2unix`).
2. کلون کردن پروژه از GitHub.
3. تنظیم و فعال‌سازی سرویس‌های مرتبط با پروژه.

## بررسی وضعیت سرویس

پس از نصب، می‌توانید وضعیت سرویس را با دستور زیر بررسی کنید:

```bash
sudo systemctl status run_update_loop.service
```

اگر نیاز به توقف یا راه‌اندازی مجدد سرویس داشته باشید، از این دستورات استفاده کنید:

- **توقف سرویس**:
  ```bash
  sudo systemctl stop run_update_loop.service
  ```

- **راه‌اندازی مجدد سرویس**:
  ```bash
  sudo systemctl start run_update_loop.service
  ```

## نتیجه‌گیری

این پروژه به شما کمک می‌کند تا قوانین V2bx مربوط به ایران را به صورت خودکار مدیریت و به‌روزرسانی کنید. با استفاده از این پروژه، نیاز به مدیریت دستی قوانین و سرویس‌ها را نخواهید داشت و همه چیز به صورت خودکار انجام می‌شود.
